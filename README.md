# Python-SQL-Walmart-Sales-Analysis

## Project Overview

Developed an end-to-end retail sales analytics project using **Python** and **PostgreSQL** to analyze **10,051 Walmart sales transactions**. The project involved cleaning raw sales data, preparing a high-quality analytical dataset, and performing SQL-based business analysis to uncover insights into customer purchasing behavior, branch performance, product profitability, payment preferences, and revenue trends.

---

## Business Problem

Retail organizations process thousands of transactions daily, making it difficult to identify customer buying patterns, high-performing product categories, branch performance, and revenue trends through manual analysis. This project transforms raw transactional data into actionable business insights that support operational and strategic decision-making.

---

## Project Objectives

- Clean and validate **10,051** retail sales records.
- Improve data quality before business analysis.
- Analyze customer purchasing behavior across multiple branches.
- Evaluate branch and product category performance.
- Compare revenue performance across different years.
- Generate business insights to support data-driven decisions.
